# ee273

9 Text-Based RPG
This project focuses on designing an interactive narrative system with structured gameplay mechanics.
The system must model characters, environments, and combat interactions within a cohesive world.
Emphasis is placed on extensible entity design and state progression.
Develop a role-playing game with locations, characters, and turn-based combat.

9.1 Core Requirements
The system must implement a world map with navigation, model player and enemy entities, support
inventory and quest tracking, and provide save and load functionality.

9.2 Optional Extensions
Extensions may include branching narrative logic or more advanced combat behaviours.

9.3 Object-Oriented Design Expectations
A polymorphic Entity hierarchy is required. Combat resolution should use strategy patterns, and STL
containers must manage inventories and quests.
